Responsive

  • Multiples Resoluciones
  • Funcionamiento de las Mediaqueries

En CSS, los responsive (o diseño responsivo) se refieren a la técnica de adaptar una página web para que se vea y funcione correctamente en diferentes tamaños de pantalla, como celulares , tables o computadoras

Multiples Resoluciones

Los llamados media queries son una serie de funciones que incorpora CSS3 para que las páginas web se adapten "racionalmente" o adopten una maquetación diferente según la resolución en la que se muestre la página web, ya sea en navegadores que funcionan sobre ordenadores con grandes monitores, sobre tablets, móviles o incluso displays de neveras, yogurteras, coches o lavadoras conectadas a Internet.

Hace algunos años las resoluciones de pantalla eran fáciles de controlar, pero hoy existen muchos dispositivos con diferentes tamaños. Por eso se agrupan las resoluciones en grupos estándar para facilitar el diseño responsive en la cual son las siguientes resoluciones:

  • Resoluciones iguales o menores de 480 píxeles: para móviles
  • Resoluciones de 481px a 768 píxeles: para tablets
  • Resoluciones mayores de 768 píxeles: para ordenadores ¿Dónde colocarlos?

Se deben de seguir los siguientes pasos:


  1. Primero colocamos todos los selectores habituales de la página, ya sea dentro del o bien dentro un archivo CSS externo.
  2. Después de todos los selectores CSS habituales se colocan los mediaqueries con sus selectores CSS dentro.
                <head>	
                <style type="text/css">

                body{
                    background-color:white;
                    font-size:14px;
                }	

                #bloque1{
                    font-size:15px;
                }

                @media only screen and (max-width:320px){
                    body{
                    background-color:red;
                    }
                    #bloque1{
                    font-size:12px;
                    }
                }

                </style>
                </head>

Funcionamiento Mediqueres


Un mediaquerie agrupa diferentes selectores CSS e identifica en qué condiciones se deben ejecutar dichos selectores. Normalmente estas condiciones están relacionadas con la resolución horizontal del dispositivo donde se está visualizando la página web.

Sintaxis


Se obtiene la siguiente sintaxis:

                @media only screen and (max-width:320px){
                
                }

                @media only screen and (min-width:980px){
                
                }
  • max-width: Equivale a la traducción "cuando la resolución horizontal sea menor que la indicada...".
  • min-width: Equivale a la traducción "cuando la resolución horizontal sea mayor que la indicada...".

Ejemplo codigo final :


                    @media only screen and (min-width:320px) and (max-width:480px){
                    
                    }

Exclusivo para Moviles y Tablets


En un móvil no tenemos una ventana del navegador que podamos redimensionar como ocurre con un ordenador, sino que toda la pantalla del móvil es utilizada como ventana del navegador. Para evitar estos problemas se realiza lo siguiente: 

                @media only screen and (min-device-width:320px) and (max-device-width:480px){
                
                }

  • Añadiendo device podemos hacer referencia -no al ancho de la ventana del navegador (que no existe como tal)- sino a la anchura real del dispositivo.
  • Para que el contenido no se escale automáticamente al visualizar una página web debemos utilizar un metadata que colocaremos dentro del para configurar el viewport con varias propiedades y que únicamente funcionará para móviles y tablets:
    •                     <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=2">

Al mismo tiempo con initial-scale=1 estamos configurando la escala inicial de visualización. Así, utilizando un valor de "1", el contenido se verá al 100% del tamaño (con "2" la escala inicial sería el doble, pudiéndose utilizar decimales "0.75", "1.25"...) Por último, podemos indicar el máximo grado de zoom que podrá hacer el usuario utilizando maximum-scale=2. En este ejemplo, con un valor de "2", se permite al usuario ampliar el contenido como máximo el doble de su tamaño original.
