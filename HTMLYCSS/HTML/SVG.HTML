SVG

  • SVG
  • Etiquetas

Un SVG en CSS y HTML es un formato de imagen vectorial que permite dibujar figuras (líneas, círculos, rectángulos, caminos, etc.) usando código, y que puede ser estilizado o animado con CSS.

SVG

Con el elemento SVG (Scalable Vector Graphics) podemos crear imágenes vectoriales redimensionables de manera dinámica o interactiva utilizando diferentes atributos. Al convertirse en 2001 en una recomendación del W3C ya es aceptado en la mayoría de navegadores (Firefox desde la versión 1.5, Opera desde su versión 8, Chrome y Safari de manera conjunta, y también increiblemente Internet Explorer, desde su versión 9).


Ejemplo de SVG:

                <svg width="150" height="90" xmlns="http://www.w3.org/2000/svg">
                <rect width="150" height="90" fill="red" stroke="black" stroke-width="1"/>
                <rect x="40" y="33" width="70" height="24" fill="white"/>
                <rect x="63" y="10" width="24" height="70" fill="white"/>
                </svg>

Etiquetas


  • SVG: El código correcto debe estar dentro de la etiqueta svg.
    •             <svg width="300" height="200" xmlns="http://www.w3.org/2000/svg">
            </svg>
  • rect: Permite crear rectángulos o cuadrados.
    •             <rect x="0" y="0" rx="12" ry="12" width="60" height="60"
            fill="gold" stroke="green" stroke-width="1" opacity="0.8"/>

Resultado:

  • circle: Permite crear círculos.
    •             <circle cx="60" cy="50" r="30" fill="olive"
            stroke="black" stroke-width="2"/>

Resultado:

  • ellipse: Permite crear elipses.
    •             <ellipse cx="80" cy="50" rx="60" ry="25"
            fill="purple" stroke="black"/>

Resultado:

  • line: Permite crear líneas.
    •             <line x1="10" y1="10" x2="150" y2="60"
            stroke="fuchsia" stroke-width="4"/>

Resultado:

  • polyline: Permite crear líneas conectadas.
    •             <polyline points="20,80 40,40 60,80 80,40 100,80"
            fill="none" stroke="blue" stroke-width="3"/>

Resultado:

  • polygon: Permite crear figuras cerradas.
    •             <polygon points="50,0 82,100 0,37 100,37 20,100"
            fill="gold" stroke="black" stroke-width="3"/>

Resultado:

  • text: Permite escribir texto dentro del SVG.
    •             <text x="30" y="40"
            fill="red" stroke="white" stroke-width="1">
