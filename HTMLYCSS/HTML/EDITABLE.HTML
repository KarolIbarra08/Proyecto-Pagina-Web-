Contenido Editable

  • Contenido Editable

En CSS/HTML, el contenido editable se refiere a elementos de una página web que pueden ser modificados directamente por el usuario desde el navegador, sin necesidad de formularios o programación adicional.

Contenido Editable

Para que un contenido sea editable (es decir, para que el usuario pueda modificar, eliminar o añadir texto) desde el propio navegador, dicho texto tiene que estar ubicado dentro de un

o de una etiqueta semántica (article, section...) con la propiedad 'contenteditable=true': 
                <section id="editable" contenteditable="true">
                <h2>Escribe tu pedido:</h2>
                    <ul>
                        <li>2 bolsas de patatas</li>
                        <li>1 dónut</li>
                        <li>4 Kg. de boniatos</li>
                    </ul>
                </section>

Si añades más elementos de los que caben dentro de la lista aparecerá un scroll vertical, que se produce gracias a este código CSS: 


                #editable{
                overflow:auto;
                }

Para recuperar las modificaciones o cambios se realiza por el usuario utilizando javaSript o jQuery.

                <button onclick="alert(document.getElementById('ejemplo').innerText)">
                Leer el contenido
                </button>

Escribe tu pedido:
