Evolución de los Sistemas Web
Web Estática (años 90)

En los primeros sitios web, las páginas eran estáticas, es decir, el contenido casi no cambiaba.

  • Características:
    1. Solo mostraban texto e imágenes.
    2. No había mucha interacción.
    3. Cada página se escribía manualmente.
  • Herramientas del frontend usadas:
    1. HTML
    2. CSS
    3. JavaScript
Web Dinámica (2000–2010)

En esta etapa las páginas empezaron a responder a las acciones del usuario sin tener que recargar todo el sitio.

  • Características:
    1. Formularios interactivos.
    2. Animaciones básicas.
    3. Validaciones en el navegador.
  • Herramientas del frontend usadas:
    1. JavaScript (más utilizado).
    2. jQuery → facilitó manipular el DOM y animaciones.
    3. AJAX → cargar datos sin recargar la página.
Aplicaciones Web Modernas (2010–2020)

Las páginas web se volvieron aplicaciones completas, similares a programas de escritorio.

  • Características:
    1. Interfaces más rápidas.
    2. Componentes reutilizables.
    3. Arquitecturas más organizadas.
  • Frameworks y herramientas frontend:
    1. React
    2. Angular
    3. Vue.js
  • Otras herramientas importantes:
    1. Bootstrap → diseño responsive.
    2. Sass → CSS más avanzado.
Web Moderna (2020–Actualidad)

Hoy el frontend es mucho más complejo y potente.

  • Características:
    1. Aplicaciones muy rápidas.
    2. Renderizado optimizado.
    3. Experiencias similares a apps móviles.
  • Herramientas modernas:
    1. Next.js → optimización y renderizado híbrido.
    2. Vite → desarrollo más rápido.
    3. Tailwind CSS → diseño con clases utilitarias.
    4. TypeScript → JavaScript con tipado.
