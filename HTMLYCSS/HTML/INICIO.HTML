Tipos de tecnologías web

HTML5 significa HyperText Markup Language versión 5. Es el lenguaje estándar que se usa para crear la estructura de las páginas web.

Características

  • Usa etiquetas semánticas como header, section, article, footer.
  • Permite integrar audio y video.
  • Es compatible con dispositivos móviles.
  • Permite usar formularios más avanzados.
  • Funciona junto con CSS y JavaScript.

Importancia

  • Es la base de todas las páginas web.
  • Mejora la organización del contenido.
  • Permite crear sitios más modernos e interactivos.
  • Facilita el posicionamiento en buscadores (SEO).

Estructura Semántica Principal

Las etiquetas semánticas ayudan a definir claramente la estructura de una página web, permitiendo que los navegadores, buscadores y lectores de pantalla comprendan mejor el contenido.

  • <header>: Define el encabezado de una página o sección. Normalmente contiene el logo, el título o el menú de navegación.
  • <nav>: Contiene los enlaces de navegación del sitio. Ayuda a los usuarios y a los lectores de pantalla a moverse entre las páginas.
  • <main>: Representa el contenido principal del documento. Solo debe existir uno por página.
  • <section>: Agrupa contenido relacionado que comparte una misma temática.
  • <article>: Se usa para contenido independiente como noticias, entradas de blog o publicaciones.
  • <aside>: Contenido relacionado indirectamente como barras laterales, publicidad o enlaces adicionales.
  • <footer>: Define el pie de página que contiene información como autor, derechos de autor, contacto o redes sociales.

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje que se usa para dar diseño y apariencia a una página web creada con

Características

  • Permite cambiar colores, fuentes y tamaños.
  • Controla el diseño y la posición de los elementos.
  • Usa selectores para aplicar estilos.
  • Permite animaciones y efectos visuales
  • Se puede reutilizar en varias páginas.

Importancia

  • Hace que las páginas web sean visualmente atractivas.
  • Separa el diseño del contenido.
  • Permite crear sitios web más organizados y profesionales.
  • Mejora la experiencia del usuario.

HTML5 y CSS trabajan juntos para crear páginas web completas. HTML5 se encarga de la estructura del contenido, mientras que CSS se encarga del diseño y la apariencia visual.

¿Cómo trabajan juntos?

  • HTML5 crea títulos, párrafos, imágenes y secciones.
  • CSS aplica colores, tamaños y posiciones.
  • Permite que el contenido tenga un diseño atractivo.
  • Juntos crean páginas modernas y adaptables.

Ejemplo sencillo

            <h1>Hola Mundo</h1>
            
            h1{
            color: blue;
            text-align: center;
            }
