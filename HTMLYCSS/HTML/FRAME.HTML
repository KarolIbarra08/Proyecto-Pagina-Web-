Frame

  • I-Frame
  • Google Maps

En HTML y CSS, un iframe (Inline Frame) es un elemento que permite mostrar otra página web o contenido externo dentro de una página web. Se utiliza para incrustar contenido como videos, mapas, documentos u otras páginas dentro de un espacio específico de la página.

I-Frame

Es una etiqueta de HTML bastante antigua que permite añadir de una manera muy fácil y rápida una página web o contenido dentro de otra página. Para ello únicamente hay que indicar la dirección web del contenido o página HTML que quieres insertar y el espacio de la ventana que mostrará dicho contenido.


Aunque en HTML5 existen alternativas a <iframe>, todavía se usa mucho porque servicios como Google Maps o YouTube lo utilizan para insertar mapas o videos en una página web. Solo es necesario copiar el código que dan estas páginas y pegarlo dentro de un div, al que luego se le puede aplicar estilos con CSS.

Ejemplo de Frames:

            <iframe width="640" height="510" src="http://www..." frameborder="0">
            </iframe>

  • width y height: definen el tamaño de la ventana donde se mostrará el contenido.
  • frameborder: permite quitar o mostrar el borde del marco.
  • src: indica la página o contenido que se mostrará dentro del iframe.
  • Para insertar un video de YouTube se explica en el tema de video del módulo 4.
  • Para agregar un mapa de Google Maps, se explica en la siguiente sección.

Google Maps

  • Accedemos a Google Maps y buscamos la dirección que queremos mostrar.
  • Ajustamos el zoom según lo que queramos mostrar.
  • Seleccionamos la opción Compartir y luego Insertar mapa.
  • Elegimos el tamaño del mapa.
  • Copiamos el código iframe que nos proporciona Google Maps.
  • Lo pegamos dentro de un div para poder modificarlo posteriormente con CSS.

Ejemplo de código:

            <iframe 
            width="640" 
            height="510" 
            src="https://www.google.com/maps?q=Popayán&output=embed" 
            frameborder="0">
            </iframe>
