Página Web (Nivel General)
Definición Una página web es un documento digital que se visualiza a través de un navegador como Google Chrome o Mozilla Firefox. Contiene información en formato de texto, imágenes, videos y enlaces, y forma parte de un sitio web con el propósito de informar, comunicar o permitir interacción con los usuarios.
Función Su función principal es informar, comunicar o permitir la interacción con los usuarios, ofreciendo contenido digital organizado y accesible desde cualquier parte del mundo.
Importancia
  • Permite compartir información a nivel mundial.
  • Facilita la comunicación entre personas y empresas.
  • Funciona como medio de promoción y venta.
  • Representa la identidad digital de una organización.
  • Brinda acceso rápido y organizado a contenidos digitales.
Características
  • Está desarrollada principalmente en HTML.
  • Se accede mediante una URL.
  • Puede ser estática o dinámica.
  • Permite integrar elementos multimedia.
  • Puede conectarse a bases de datos.
  • Es accesible desde cualquier dispositivo con internet.

Arquitectura Web

La arquitectura web es la estructura general que define cómo interactúan todos los componentes de una aplicación web: cliente, servidor, bases de datos y servicios externos.

Desde un punto de vista técnico, establece:

  • Cómo el navegador se comunica con el servidor mediante HTTP/HTTPS.
  • Cómo se procesan las solicitudes y respuestas.
  • Cómo se organiza el backend (controladores, servicios, modelos).
  • Cómo se integran APIs o microservicios.

Arquitectura Frontend

La arquitectura frontend se enfoca específicamente en la organización interna de la parte visual e interactiva que ve el usuario en el navegador.

Desde un punto de vista técnico, establece:

  • Cómo se estructuran los archivos (HTML, CSS, JavaScript).
  • Cómo se organizan los componentes.
  • Cómo se maneja el estado de la aplicación.
  • Cómo se comunican los módulos entre sí.
  • Qué patrón de diseño se utiliza (ejemplo: MVC).
