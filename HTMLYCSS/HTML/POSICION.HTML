Posicionamiento de elementos

  • Posicionamiento relative
  • Posicionamiento absolute
  • Posicionamiento fixed

El posicionamiento de cajas en CSS es la forma en que se ubican o colocan los elementos (como div, section, article, etc.) dentro de una página web. Estas “cajas” forman parte del modelo de caja de CSS y se pueden mover o acomodar en diferentes lugares de la pantalla.

Posicionamiento de cajas

Existen tres opciones de posicionamiento de las cajas dentro del body.

  • Relative:Es el posicionamiento por defecto. Las cajas se colocan en relación con las demás y si una cambia de tamaño o posición, las otras se mueven. No permite colocar una caja encima de otra.
  • Absolute:Permite colocar una caja en una posición exacta sin depender de las otras cajas. Con este tipo sí se puede poner una caja encima de otra.
  • Fixed:Funciona parecido al absoluto, pero la caja se queda fija en la pantalla aunque se haga scroll en la página.

Ejemplo de posición de cajas:

            #caja_relative{
                position: relative;
                top:20px;
                left:30px;
            }

            #caja_absolute{
                position: absolute;
                top:40px;
                left:150px;
            }

            #caja_fixed{
                position: fixed;
                bottom:20px;
                right:20px;
            }

Importante

Al definir el posicionamiento como absolute o como fixed es necesario indicar la posición o coordenada exacta en la que se desea ubicar la caja (o div).


Para ello podemos utilizar los siguientes valores:

  • Left:Especifica la coordenada horizontal, empezando a contar desde el extremo izquierdo.
  • Righ:Especifica la coordenada horizontal, empezando a contar desde el extremo derecho.
  • Top:Especifica la coordenada vertical, empezando a contar desde el extremo superior.
  • Bottom:Especifica la coordenada vertical, empezando a contar desde el extremo inferior..

Cuando se usa posición absolute, el elemento puede cambiar de lugar si cambia el tamaño de la ventana del navegador. Para evitar este problema, se recomienda colocar el

con posición absoluta dentro de un contenedor con posición relative, así su posición se mantiene correcta.

Ejemplo de posición de cajas:

                #contenedor{
                position:relative;
                }

                #div_absoluto{
                position:absolute;
                }
