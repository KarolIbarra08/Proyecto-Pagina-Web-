Imagenes de Fondo

  • Palabras Magicas
  • Superposicion
  • Extras

En CSS, las imágenes de fondo son imágenes que se colocan detrás del contenido de un elemento HTML (como un div, body, section, etc.) para decorar o mejorar el diseño de una página web.

Palabras Magicas

En HTML5 existen propiedades CSS que permiten añadir fondos de página, ya sea con colores, imágenes o repeticiones de imágenes.

Color de fondo

        background-color: rgba(255,255,0,0.4);

Superoposicion de Fondos

Como novedad, en HTML5, ahora tenemos la posibilidad de construir un fondo de pantalla o de un div superponiendo diferentes imágenes individuales (como si fueran capas en Photoshop).

Ejemplo imagenes:

                body{
                background:url('../img/css.png'), url('../img/rayas3.png'), url('../img/infantil.gif');
                }
  • Se utilizan tres imágenes superpuestas para crear un fondo. El orden en que se colocan es importante, porque define cuál imagen queda encima y cuál debajo.
  • La primera imagen se muestra sobre las demás y la última queda al fondo. Para que las imágenes que están debajo se puedan ver, es necesario que las imágenes superiores tengan transparencia.
  • En el ejemplo, el fondo infantil es opaco, el escudo está en una imagen con fondo transparente, y el fondo azul es semitransparente, lo que produce un efecto azulado sobre el fondo infantil.

Extras

  • Tener una imagen de fondo fija, que no se moverá con el contenido de la página.

    • Ejemplo de imagen:

            background:url('img/fondo.jpg');
        background-attachment:fixed;

    Resultado:

  • Tener una imagen adaptable de fondo que se adaptará al espacio del div donde se ubique.

    • Ejemplo de imagen:

            background:url('img/fondo.jpg');
        background-repeat:no-repeat;
        background-size:cover;

    Resultado:
