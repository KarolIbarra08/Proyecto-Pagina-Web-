En CSS, una transición es un efecto que permite que un cambio en una propiedad de estilo ocurra de manera gradual y suave, en lugar de cambiar de forma inmediata o brusca

Tansiciones

Las transiciones (transition) funcionan como pequeñas animaciones que siguen las mismas características que la etiqueta 'transform' (cambio de posición, tamaño, rotación...) aunque con la capacidad de poder especificar una duración para que se produzca dicha transformación y así poder utilizarla como una animación sencilla.


A este div le vamos a aplicar varias transiciones (de posición, tamaño y rotación) y para que podamos activar cada una de ellas en el momento que queramos deberás pasar el cursor por encima de cada ejemplo.


  • webkit- (para Chrome y Safari)
  • moz- (para Firefox).
  • o- (para Opera)
  • ms- (para las versiones modernas de Internet Explorer)

Transicones Con Una Unica Propiedad

Cuando pases con el cursor por encima del martillo que hay bajo este texto se producirá una transición que durará 1 segundo y medio y queproducirá que el

martillo se desplace horizontamente 400 píxeles hacia la derecha.


Ejemplo de transiciones:

               #martillo{
                transition:left 1.5s;
                }

                #martillo:hover{
                position:relative;
                left:400px;
                }

En el código se usa un div (#martillo) que contiene el objeto que se va a mover. La transición hace que el elemento se desplace hacia la derecha durante 1.5 segundos cuando el usuario pasa el cursor sobre él (:hover). Para que el movimiento funcione se cambia el valor de left, y por eso es necesario que la propiedad position sea absolute o relative

Escala

Para especificar el cambio de varias propiedades con diferentes tiempos para para cada una de ellas, únicamente tenemos que separar las diferentes propiedades con ",".


Así, en este ejemplo, al pasar el cursor por encima, se producirán 6 cambios de propiedades al mismo tiempo


  • El div martillo se moverá 400px hacia la derecha (utilizando 'left') en 1.5 segundos.
    • .
  • El color de fondo ('background-color') del
    martillo se convertirá en negro en 2 segundos.
  • La anchura del div se modificará hasta llegar a los 225 píxeles en 1 segundo.
  • La altura del se modificará hasta llegar a los 200 píxeles (empujando hacia abajo el contenido que le sigue) durante 1 segundo.
  • El espacio que hay diventre el extremo izquierdo del
    y el dibujo que éste contiene ('padding-left') se expandirá hasta llegar a los 100 pixeles en 1 segundo.
  • El espacio que hay entre el extremo superior del
    y el dibujo que éste contiene ('padding-top') se expandirá hasta llegar a los 20 pixeles en 1 segundo.

Ejemplo de transiciones:

                #martillo{
                background-color:orange;
                width:100px;
                height:175px;
                transition:left 1.5s, background-color 2s, width 1s, height 1s, padding-left 1s, padding-top 1s;
                }

                #martillo:hover{
                position:relative;
                left:400px;
                background-color:black
                width:225px;
                height:200px;
                padding-left:100px;
                padding-top:20px;
                }

En el primer selector (#martillo) se especifican los valores iniciales (color de fondo 'background-color', anchura 'width' y altura 'height'), los que no se definan (como 'padding-left' y 'padding-top') se tomarán los valores por defecto.

Palabras Magicas

Aunque no es obligatorio, tenemos la posibilidad de especificar el tipo de movimiento y si debe existir algún retardo para que empiece la animación.


Además, como todavia es necesario repetir el código anteponiendo los prefijos -webkit-, -moz-, -op- y -ms- (para asegurar la compatibilidad con todos los navegadores), es interesante poder especificar todos estos parámetros en una sola línea.

Elemento Descripción Ejemplos
Propiedades a modificar Son las propiedades CSS que cambiarán durante la transición. all, background-color, border, border-radius, color, top, bottom, left, right, box-shadow, width, height, line-height, margin, opacity, word-spacing, letter-spacing, fill, padding, stroke, text-shadow, vertical-align, visibility, z-index
Duración en segundos Indica cuánto tiempo durará la animación. Se escribe el número seguido de la letra s. 1s, 2s, 3s
Tipo de animación Define la velocidad del movimiento durante la transición. linear, ease, ease-in, ease-out, ease-in-out
Retardo Tiempo que el navegador espera antes de comenzar la animación. 0.5s, 1s, 2s

Ejemplo de transiciones:

                transition:[propiedad a modificar] [Duración en segundos] [Tipo de animación] [Retardo];
                transition: width 2s ease-out, 0.5s;
                

                #martillo{
                opacity:1;
                transition:opacity 2s ease-out 0.5s;
                }

                #martillo:hover{
                opacity:0;	
                }

Cuando coloques el cursor encima del martillo que hay sobre estas líneas, primero -se esperará medio segundo (0.5s)-, después de la espera, el martillo se irá haciendo transparente en una animación con deceleración (ease-out) que durará 2 segundos (empezará muy rápido y a medida que vaya haciendose transparente irá reduciendo la velocidad).

Compatibilidad de navegadores

Para garantizar la compatibilidad con navegadores antiguos, las transiciones es conveniente que lleven los prefijos, quedando de esta manera un código más largo que la deuda exterior de Zaire.


Ejemplo de transformacion:

               #martillo{
                opacity:1;
                -webkit-opacity:1;
                -moz-opacity:1;
                -o-opacity:1;
                -ms-opacity:1;

                transition:opacity 2s ease-out 0.5s;
                -webkit-transition:opacity 2s ease-out 0.5s;
                -moz-transition:opacity 2s ease-out 0.5s;
                -o-transition:opacity 2s ease-out 0.5s;
                -ms-transition:opacity 2s ease-out 0.5s;
                }

                #martillo:hover{
                opacity:0;	
                -webkit-opacity:0;
                -moz-opacity:0;
                -o-opacity:0;
                -ms-opacity:0;
                }
