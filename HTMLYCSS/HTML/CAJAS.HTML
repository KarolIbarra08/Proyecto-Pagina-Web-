Cajas Flotantes

  • Uso del float
  • Ejemplo del float
  • Clean del float

En CSS, las cajas flotantes son elementos a los que se les aplica la propiedad float para que se desplacen hacia la izquierda o hacia la derecha dentro de su contenedor, permitiendo que otros elementos (como el texto) se coloquen alrededor de ellos.

Usos del Float

La propiedad float sirve para alinear cajas o a la izquierda o a la derecha de otras cajas, utilizando el valor float:left o float:right.
Al usar float hay que tener claro una serie de cuestiones:

  • Para alinear varias cajas (o divs) una a la derecha de la otra, todas las cajas tienen que tener un width determinado y un float.
  • La suma de todos los width de todas las cajas deben caber en el ancho del . Si no caben se pondrá una debajo de la otra.
  • Por lo general, en todas las cajas se utiliza el mismo tipo de alineación (o todas left o todas right).
  • Aunque se quiera colocar una caja a la derecha de otra, se utiliza float:left, ya que el valor right indica que las cajas se alinearán desde la parte derecha de la ventana del navegador.
  • La siguiente caja que venga después de la última caja que utiliza un float, (aunque no tenga ningún valor float), se alineará en el espacio vacio que deje la alineación anterior. Para evitar esto, a esta nueva caja se le debe añadir la declaración: clear:both;. De esta manera no aprovechará el espacio vacio que pudiera quedar y se colocará correctamente debajo de la última caja.

Ejemplo Practico

Tenemos dos cajas (divs), etiquetadas con los id="caja1" e id="caja2". La primera (#caja1) contiene una imagen de 200 x 133 píxeles, mientras que #caja2 contiene un texto centrado verticalmente.

Ejemplo CSS de cajas

Para que ambas cajas no se sitúen una debajo de la otra (como es lo habitual), tenemos que añadir un width y un float a cada caja: 

                #caja1{
                    width:200px;
                    float:left;
                    margin-right:15px;
                    }
                    #caja2{
                    width:200px;
                    float:left;
                    }

Este es un ejemplo de texto dentro de la segunda caja. Ambas cajas están alineadas usando la propiedad float en CSS.

Clear:Both

La mayoría de problemas de colocación que se producen al utilizar la propiedad float se pueden solucionar de dos maneras:

Tipos de clear :

Ejemplo CSS de diferentes clear:

  • Ejemplo uno: 
                        #caja_descolocada{
                        clear: both;
                    }

    Utilizando clear:both es una solución sencilla pero muy efectiva. Si a la primera caja que viene después de un elemento con float se le aplica clear:both, no aprovechará el espacio libre y se colocará debajo de la caja anterior.

  • Ejemplo dos: 
                        <div id="cajas">
                        <div id="caja1">
                        </div>

                        <div id="caja2">
                        </div>
                    </div>

    A esta estructura HTML le tenemos que añadir los correspondientes selectores CSS: 


                       #cajas{
                    height:131px;
                    }
                    #caja1{
                    width:395px;
                    float:left;
                    margin-right:15px;
                    }
                    #caja2{
                    width:300px;
                    float:left;
                    }

Este es un ejemplo de texto dentro de la segunda caja. Ambas cajas están alineadas usando la propiedad float en CSS.
