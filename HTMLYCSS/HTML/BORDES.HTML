Bordes y bordes redondeados

  • Líneas separadoras horizontales
  • Líneas separadoras verticales
  • Bordes simples
  • Bordes redondeados

En CSS, los bordes redondeados son una propiedad que permite que las esquinas de un elemento no sean rectas sino curvas mediante border-radius, lo que mejora la apariencia de cajas, imágenes o botones. Por otro lado, las líneas se utilizan para separar o decorar contenido en una página web y se pueden crear con la etiqueta hr de HTML5 o con propiedades de borde como border.

Líneas separadoras horizontales

Para crear una línea horizontal la forma fácil y rápida es utilizar la etiqueta <hr>.

  • border-width (grosor)
  • width (anchura)
  • border-color (color)
  • border-style (tipo de línea)

Ejemplo de selector CSS para una línea específica:

                    #linea1{
                    border-width:4px;
                    border-style:dotted;
                    border-color:blue;
                    width:50%;
                    }

Líneas separadoras verticales

Para crear líneas verticales definimos uno de los bordes de una caja <div>.

  • border-right
  • border-left

Ejemplo de línea vertical:

                .linea-vertical{
                height:80px;
                border-right:4px dashed #3C5391;
                }

Bordes simples

Podemos aplicar bordes a cualquier elemento utilizando la propiedad border.

Tipos de bordes:

  • none: sin línea
  • solid: línea continua
  • dotted: línea punteada
  • dashed: línea discontinua
  • inset: bajorelieve
  • outset: relieve

Ejemplo CSS de diferentes bordes:

                .borde-none{
                border:3px none black;
                }

                .borde-solid{
                border:3px solid black;
                }

                .borde-dotted{
                border:3px dotted red;
                }

                .borde-dashed{
                border:3px dashed blue;
                }

                .borde-inset{
                border:5px inset gray;
                }

                .borde-outset{
                border:5px outset green;
                }

Bordes redondeados

Con la propiedad border-radius suavizamos las esquinas.

Ejemplo de bordes redondeados:

                #apartado1{
                width:250px;
                height:100px;
                border:3px solid brown;
                border-radius:22px;
                }

                #apartado-mixto{
                width:250px;
                height:100px;
                border:3px solid #3C5391;
                border-radius:10px 0px 39px 22px;
                }

Resultado:

border-radius:22px

border-radius:10px 0px 39px 22px
