Sombras

  • Sombras cajas
  • Creacion de halos o resplandores
  • Sobra de texto
  • Multiples sombras

En CSS, las sombras son efectos visuales que se usan para crear la apariencia de profundidad o relieve en los elementos de una página web. Permiten que un objeto parezca elevado o separado del fondo, agregando una sombra alrededor del texto o de una caja.

Sombras Div o Cajas

Con HTML5 podemos crear de una manera sencilla tanto sombras para zonas del HTML <div, article, footer, header >. Aunque cada uno de estos tipos de sombra tiene su propia etiqueta, la sintaxis sigue la misma lógica.

Ejemplo CSS de diferentes sombras:

                    section{
                    box-shadow: 6px 20px 10px black;
                    }
  • El primer valor (6px), indica el número de píxeles que se desviará la sombra horizontalmente.
  • El segundo valor (20px), especifica el número de píxeles que se moverá la sombra verticalmente.
  • El tercer valor (10px), es la suavidad que tendrá la sombra (a menor valor más duro será el borde de la sombra).
  • El último valor pertenece al color de la sombra (indicando el nombre de color o su código hexadecimal).

Sombras Internas

Para crear sombras internas únicamente tenemos que añadir el valor inset al final de <box-shadow>.

Ejemplo CSS de diferentes sombras::

                section{
                box-shadow: 7px 9px 15px black inset;
                }

Halos o Resplandores

Para crear un halo o resplandor que rodee una caja o el

que corresponda indicamos '0px' en los dos primeros valores y especificamos el color del resplandor.

Tipos de bordes:

Ejemplo CSS de diferentes sombras:

                section{
                box-shadow: 0px 0px 30px blue;
                }

Sombra de Textos

En el caso de las sombras en textos utilizamos la etiqueta 'text-shadow' que funciona exactamente igual que 'box-shadow':

Ejemplo en CSS de sombras:

                #apartado1{
                text-shadow: 10px 10px 5px gray;
                }
  • El primer valor (10px) es la separación horizontal de la sombra (hacia la derecha, ya que es positivo).
  • El segundo valor (10px) es la separación vertical de la sombra (hacia abajo, ya que es positivo).
  • El tercer valor (5px) es la suavidad de la sombra.
  • El último valor (gray) es el color de la sombra.

Multiples Sombras

Para aplicar más de una sombra a un mismo elemento debemos añadir los diferentes parámetros de cada una de las sombras separadas por comas:

Ejemplo en CSS de sombras:

                #apartado1{
                box-shadow: 0px 0px 20px red inset, 0px 0px 25px blue;
                }

La primera sombra (0px 0px 20px red inset) está situada justo encima del div, con una distancia de sombra de 20 píxeles, de color roja e interna. La segunda (0px 0px 25px blue) también está situada justo encima del div, con una longitud de sombra de 25 píxeles, de color azul y como no se indica ("inset"), es una sombra externa.
